# Line Lyhne Inspired Pelican Theme

A typography-first editorial Pelican theme inspired by minimalist Scandinavian portfolio/blog design.

## Features

- Elegant serif typography
- Responsive layout
- Dark mode
- Minimal CSS
- Quiet editorial aesthetic
- Large article titles
- Lightweight and static

## Installation

Extract into your Pelican themes directory and set:

```python
THEME = "themes/linelyhne-inspired"
```

Example:

```python
SITENAME = "Death By Boat"

MENUITEMS = (
    ("about", "/pages/about.html"),
    ("archive", "/archives.html"),
)
```
