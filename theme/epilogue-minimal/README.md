# Epilogue Minimal Pelican Theme

Minimal typography-first Pelican theme inspired by the layout and typography style of linelyhne.com.

## Features

- Epilogue font
- minimalist layout
- responsive design
- dark mode support
- lightweight/no JavaScript
- large understated typography

## Installation

Extract into your Pelican themes directory:

```python
THEME = "themes/epilogue-minimal"
```

Example:

```python
SITENAME = "Death By Boat"

MENUITEMS = (
    ("about", "/pages/about.html"),
    ("archive", "/archives.html"),
)
```
